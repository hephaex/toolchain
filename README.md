Toolchain
=========

OSX opensource toolchain

## prerequirement
1. xcode install
> $ xcode-select --install

# cmake
> $ ./cmake_install.sh

## autoconf & automake & libtool & texinfo
> $ ./autoconf_install.sh

## emacs 25.2.1 no-x-windows 
> $ ./osx_emacs25-nox.sh 
> $ alias emacs="/usr/local/bin/emacs -nw"

## openssl-1.0.1i
> $ ./openssl.sh

## ruby
> $ ./ruby.sh

## vagrant
> $ ./vagrant.sh
